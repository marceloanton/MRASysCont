# Product Context

MRASysCont es un sistema contable web multi-empresa para estudios contables.

El contador opera como super-usuario y administra multiples empresas cliente desde una sola plataforma. Cada empresa es un tenant aislado con datos contables, fiscales, documentales y operativos propios.

## Alcance inicial

- Plataforma web moderna.
- Multi-empresa con aislamiento por `empresa_id`.
- Contabilidad formal.
- Facturacion ARCA/AFIP por empresa.
- Multi-moneda ARS/USD.
- Portal cliente de lectura y subida documental.
- Caja/bancos basico.
- Reportes legales y de gestion.
