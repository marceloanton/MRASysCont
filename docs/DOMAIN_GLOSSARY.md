# Glosario De Dominio

- Estudio: tenant principal del sistema.
- Cliente del estudio: persona o entidad que contrata al estudio.
- Empresa: entidad contable/fiscal asociada a CUIT.
- Tercero: cliente comercial o proveedor de una empresa.
- Comprobante: documento comercial (factura, NC, ND, recibo).
- Asiento: registro contable por partida doble.
- Periodo contable: intervalo de registracion con estado operativo.
- Presentacion fiscal: obligacion declarativa o de pago.
- Documento: archivo respaldatorio del expediente.
- Papel de trabajo: evidencia profesional para liquidaciones/cierres.
