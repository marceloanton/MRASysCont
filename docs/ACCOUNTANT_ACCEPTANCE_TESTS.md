# Pruebas De Aceptacion Profesional

## Caso 1: Alta De Cliente Nuevo
1. Crear cliente del estudio.
2. Crear empresa/CUIT.
3. Cargar ficha fiscal.
4. Asignar responsable interno.
5. Crear vencimientos iniciales.
6. Solicitar documentacion.

Criterio: cliente visible en dashboard con estado inicial y pendientes.

## Caso 2: Cierre Mensual Simple
1. Revisar documentos.
2. Registrar ventas y compras.
3. Generar IVA base.
4. Conciliar con contabilidad.
5. Crear papel de trabajo.
6. Marcar presentacion para revision.

Criterio: periodo trazable y documentado.

## Caso 3: Correccion Contable
1. Detectar error en asiento confirmado.
2. Crear contraasiento/ajuste.
3. Ver impacto en mayor/balance.

Criterio: asiento original inmutable.
