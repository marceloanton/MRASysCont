# Onboarding Offer - MRASysCont

## Objetivo
Acelerar puesta en marcha de un estudio contable en MRASysCont.

## Alcance del onboarding
1. Alta del estudio
2. Configuracion de usuarios y permisos
3. Alta de empresas cliente
4. Carga inicial de terceros
5. Parametrizacion operativa
6. Capacitacion base (1-2 sesiones)

## Entregables
- Ambiente configurado
- Checklist de validacion inicial
- Guion de operacion mensual

## Duracion sugerida
- 1 a 3 semanas segun complejidad.

## Requisitos del cliente
- Responsable funcional
- Datos iniciales
- Validacion de permisos y alcance
