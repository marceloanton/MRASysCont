# Dangerous Actions

Acciones criticas:
- cerrar/reabrir periodo
- confirmar/reversar asiento
- emitir/anular comprobante fiscal
- cambiar permisos
- descargar masivo
- restaurar backup
- importar masivo

Reglas:
- permiso especifico
- confirmacion explicita
- auditoria
- motivo cuando corresponda
- transaccion y validaciones previas
