# FAQ - MRASysCont

## El sistema es multiempresa?
Si. El scope operativo usa `studyId` + `companyId` y, cuando aplica, `clientOfStudyId`.

## Se puede usar para facturacion real sin pagar?
No. El uso comercial/facturacion real requiere licencia comercial.

## Donde estan las reglas canonicas?
En `AGENTS.md` y docs canonicos (`TENANCY...`, `ENDPOINT_AUTHORIZATION`, `STATE_MACHINE`, etc.).

## Como validar antes de subir cambios?
Ejecutar `./scripts/prepush-check.ps1`.
