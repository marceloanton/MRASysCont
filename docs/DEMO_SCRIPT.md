# Demo Script - MRASysCont (2-3 minutos)

## Objetivo
Mostrar el flujo principal de valor para un estudio contable.

## Guion rapido

1) Inicio y contexto (15s)
- Mostrar login y dashboard.
- Explicar tenancy: estudio -> cliente -> empresa.

2) Gestion del estudio (30s)
- Entrar a clientes/tareas/vencimientos.
- Mostrar alcance por estudio y permisos.

3) Comprobantes locales (35s)
- Crear comprobante borrador.
- Confirmar con reglas (numeracion, periodo, asiento valido).

4) Contabilidad (35s)
- Ir a asientos/periodos/reportes.
- Mostrar que no se permite operar en periodo cerrado.

5) IVA base (35s)
- Mostrar libro IVA ventas/compras.
- Reporte mensual y export CSV/Excel.
- Conciliacion IVA vs contabilidad.

6) Cierre comercial (20s)
- Resumen de valor.
- Propuesta de demo privada y onboarding.

## Frase de cierre sugerida
"MRASysCont ya permite operar controladamente estudio + comprobantes + IVA con trazabilidad y aislamiento por tenant."
