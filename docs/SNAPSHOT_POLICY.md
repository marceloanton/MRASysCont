# Snapshot Policy

Documentos emitidos/publicados guardan snapshot de datos relevantes.

## Aplica A
- comprobantes
- PDFs
- informes publicados
- presentaciones fiscales
- papeles de trabajo cerrados

## Factura (minimo)
Emisor, receptor, condicion fiscal, CUIT, punto de venta, numero, CAE/QR, totales, moneda, tipo de cambio, items, alicuotas, version de plantilla.
