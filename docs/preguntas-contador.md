# Preguntas Para El Contador

## Objetivo

Reunir las preguntas necesarias para cerrar reglas de negocio, alcance fiscal, operacion diaria y prioridades antes de programar cada fase.

## Empresas y operacion

- Cuantas empresas se cargaran al inicio?
- Cuantas empresas espera administrar en 12 meses?
- Las empresas son del mismo rubro o rubros distintos?
- Cada empresa tendra usuarios propios?
- Que datos minimos necesita cargar al crear una empresa?
- Hay empresas vinculadas entre si o grupos economicos?

## Usuarios y permisos

- Quienes seran contadores administradores?
- Habra asistentes?
- Que tareas puede hacer cada asistente?
- Los asistentes se asignan por empresa, por modulo o por ambas cosas?
- Los clientes solo ven informacion o tambien cargan datos?
- Que informacion no debe ver nunca un cliente?

## Plan de cuentas

- Existe un plan de cuentas modelo?
- Hay planes por rubro?
- Que nivel de detalle necesita?
- Se permiten cuentas imputables y no imputables?
- Como se codifican las cuentas?
- Que cuentas deben bloquearse o protegerse?

## Asientos y cierres

- Quien confirma asientos?
- Se permite dejar asientos en borrador?
- Como se corrigen errores?
- En que casos se reabre un periodo?
- Como se realiza el cierre mensual?
- Como se realiza el cierre anual?
- Que reportes necesita antes de cerrar?

## Facturacion ARCA/AFIP

- Que tipos de comprobantes emite cada empresa?
- Que comprobantes son obligatorios en la primera entrega?
- Hay empresas que necesitan FCE MiPyME?
- Hay empresas exportadoras?
- Cuantos puntos de venta tiene cada empresa?
- Quien gestionara certificados y autorizaciones?
- Se empezara en homologacion con una empresa piloto?

## IVA e impuestos

- Que condiciones fiscales manejan las empresas?
- Se necesitan libros IVA compras y ventas desde v1?
- Que provincias aplican para IIBB?
- Hay retenciones o percepciones frecuentes?
- Se necesita padron o reglas por jurisdiccion?
- Que reportes fiscales entrega hoy el estudio?

## Multimoneda

- Que empresas operan en USD?
- Que operaciones se hacen en USD: ventas, compras, cobros, pagos o bancos?
- Que tipo de cambio usa el contador?
- El tipo de cambio lo carga manualmente o debe venir de una fuente externa?
- Necesita calculo automatico de diferencias de cambio?

## Portal cliente

- Que debe ver el cliente?
- Que informes se publican?
- El cliente debe subir facturas, recibos, extractos o todos esos documentos?
- Que formatos de archivo se aceptan?
- El cliente recibe observaciones?
- Se necesitan notificaciones?

## Tesoreria

- Se llevaran cajas por empresa?
- Se registraran cuentas bancarias?
- Se necesitan cheques en v1?
- Como se registran cobros y pagos hoy?
- Se hace conciliacion bancaria?

## Reportes

- Que reportes usa todos los meses?
- Que reportes entrega a clientes?
- Que reportes necesita en PDF?
- Que reportes necesita en Excel/CSV?
- Necesita comparativos entre empresas?
- Necesita dashboard general del estudio?

## Migracion

- Hay datos existentes para importar?
- En que formato estan?
- Se importaran planes de cuentas?
- Se importaran clientes/proveedores?
- Se importaran saldos iniciales?
- Se importaran comprobantes historicos?

## Comercial y mantenimiento

- El sistema sera exclusivo para este contador o producto reutilizable?
- Quien sera dueno de los datos?
- Quien pagara hosting, backups y mantenimiento?
- Que soporte se espera?
- Que pasa si cambia la normativa fiscal?
- Como se cotizan funcionalidades fuera de alcance?
