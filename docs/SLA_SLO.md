# SLA / SLO - MRASysCont

## Nota
Estos objetivos son referencia operativa y deben formalizarse en contrato comercial.

## SLO sugeridos

### Disponibilidad aplicacion
- Plan Base: 99.0%
- Plan Pro: 99.5%
- Enterprise: 99.9% (segun acuerdo)

### Soporte
- SEV1: respuesta inicial <= 2h habiles
- SEV2: respuesta inicial <= 8h habiles
- SEV3: respuesta inicial <= 24h habiles

### Backup y recuperacion
- Backup diario minimo
- Restore de prueba semanal/quincenal

## Exclusiones
- Limites de proveedores free tier
- Fuerza mayor
- Fallas por configuracion externa del cliente

## Medicion
- Ventana mensual
- Registro de incidentes y postmortem
